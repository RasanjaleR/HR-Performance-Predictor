# HR-Performance-Predictor
An end-to-end Machine Learning web application built with Streamlit and Scikit-learn to predict employee performance scores based on satisfaction, tenure and workload metrics.

🚀 Project Overview
In modern HR Management, understanding the drivers of productivity is essential. This project utilizes a large scale dataset (100,000+ records) to train a regression model that identifies how factors like Job Satisfaction, Overtime Hours and Tenure impact an employee's final performance rating.

Key Features
Real time Prediction: Input employee data via a clean sidebar interface and get immediate results.

Data Driven Insights: Uses a model trained on metrics like salary, promotions and work life balance.

Interactive UI: Built with Streamlit for a seamless, "no-code" user experience.

Scalable Pipeline: Includes full data preprocessing (handling categorical variables and scaling).

🛠️ Tech Stack
Language: Python

Libraries: Pandas, NumPy, Scikit-learn

Deployment: Streamlit

Model: Regression based Machine Learning

📊 Dataset Features
The model analyzes several key attributes, including:

Years at Company: Employee tenure.

Monthly Salary: Compensation level.

Overtime Hours: Indicator of workload and potential burnout.

Employee Satisfaction Score: Subjective well-being (1.0 - 5.0).

Promotions: Historical career growth within the company.
